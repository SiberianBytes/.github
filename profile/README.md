# SiberianBytes
Исследовательская лаборатория личных проектов, посвящённых экспериментам с:
лексическими анализаторами и токенизацией
синтаксическими парсерами и грамматиками
абстрактными синтаксическими деревьями (AST)
байткодом и виртуальными машинами
интерпретаторами и компиляторами
транспиляцией между DSL и языками общего назначения
низкоуровневой генерацией кода (x86, ARM, RISC-V)
оптимизациями на уровне промежуточного представления (IR)

🚀 Основные направления разработки
| Направление                          | Технологии                     |
|--------------------------------------|--------------------------------|
| Лексеры, парсеры, грамматики         | Rust (nom, pest), Go (text/scanner) |
| AST-трансформеры и анализ кода       | TypeScript, Python (AST module)|
| Виртуальные машины и байткод         | C, Zig, Odin                   |
| Интерпретаторы учебных языков        | Nim, Pascal, Delphi            |
| Компиляторы под целевые архитектуры  | LLVM (lite), чистый x86 asm    |
| DSL для электронных схем и сигналов  | Rust + embedded toolchains     |

🎯 Цели проекта
Изучение архитектуры компиляторов «от лексемы до машинного кода»
Создание интерпретаторов для экспериментов
Практика системного программирования через призму трансляции кода
Лаборатория для отладки понимания: как текст превращается в токены → дерево → байткод → сигналы в транзисторах
Формирование коллекции «ламповых» реализаций — без зависимостей, с прозрачной логикой
